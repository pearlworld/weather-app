<h1>☁️ Weather-App</h1>
전세계 날씨 정보 앱


## 📍 프로젝트 소개
OpenWeatherMap api를 활용해 만든 나라별로 검색과 즐겨찾기가 가능하고 실시간으로 날씨를 확인할 수 있는 

## 🌟 프로젝트 정보
- 프로젝트 : 날씨 앱<br />
- 설계 및 제작 : 최진주 <br />
- 제작기간 : 2023.09.15 ~ 2023.10.09 <br />
- 기능 : 나라별 검색, 즐겨찾기 추가·삭제
